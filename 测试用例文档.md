## 5. 技术要点

### 5.1 测试环境
- 数据库：MySQL 8.0（payclip_test，腾讯云 CynosDB）
- 每次运行前 DROP + CREATE 重建表结构
- TestClient 触发 startup 事件自动创建 super_admin
- 每个 function-scoped fixture 创建独立租户，互不干扰
- H5 移动端：Vant + Vue3 + Vite（http://localhost:5174）

### 5.2 测试策略
- **隔离测试**：创建两个独立租户，验证数据互不可见
- **状态机测试**：完整流转 active→readonly→soft_deleted→active
- **边界测试**：员工数不再有上限，验证连续创建5个都成功
- **权限测试**：tenant_admin 不能访问 super_admin 接口
- **离职即停测试**：在职 vs 离职两月对比，验证薪资记录有无
- **补回归属测试**：业务转交后，补回归属当前负责人而非原被扣人
- **窗口测试**：12月窗口内 vs 窗口外，验证销售提成有无
- **幂等测试**：已锁定月份拒绝重算 + 解锁后可重算
- **mobile E2E 测试**：覆盖 mobile 端全部页面，与桌面端功能对齐验证

### 5.3 已知限制
- H5 移动端 UI 测试为手动验证（Playwright 截图 + 真机交互）
- 支付流程为手动开通（Phase 2 接入在线支付）
- 小程序版本未开发（Phase 2）
- 计算失败回滚的异常注入测试未实现（当前代码已用 try/except 包裹计算逻辑并 rollback）

## 6. 运行测试

### 6.1 后端自动化测试

```bash
cd payclip/backend
python -m pytest tests/ -v
```

测试会自动：
1. 连接腾讯云 CynosDB MySQL 测试库 payclip_test
2. DROP + CREATE 重建所有表
3. 触发 FastAPI startup 创建 super_admin 种子
4. 为每个测试函数创建独立租户环境
5. 运行全部 31 个后端测试

### 6.2 H5 移动端 E2E 测试

```bash
cd payclip/mobile
npm run dev   # 启动 http://localhost:5174
```

手动 E2E 测试步骤：
1. 启动后端服务（`cd payclip/backend && python -m uvicorn app.main:app --reload`）
2. 启动 mobile 前端（`cd payclip/mobile && npm run dev`）
3. 浏览器打开 http://localhost:5174 进行测试
4. 按 7.x 章节用例依次验证
